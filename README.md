# **Insertion Sort Projesi Cevaplar**
### **1. Cevap** 
* *Belirtilen short türüne göre aşamalar;*
 ```
      [16,21,11,8,12,22]
         /          \
   [16,21,11]     [8,12,22]
       /   \        /     \
   [16,21] [11]  [8,12]  [22]
      / \    \     / \     \
   [16][21] [11] [8][12]  [22]
      \  /   /    \  /     /
   [16,21] [11]  [8,12]  [22]
      \     /      \      / 
   [11,16,21]     [12,18,22]
        \             / 
      [11,12,16,18,21,22]

```
*  *Big-O gösterimi:*  o(logn)
